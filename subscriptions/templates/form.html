03 — How it works

four steps · zero passwords
Footprint Six Docker containers in ~165 MB idle on an Oracle Cloud always-free 1 GB shared-CPU VM. We have headroom.

  1. You subscribe.

    Pick categories — by preset or à la carte from the 155 official arXiv codes. Optionally narrow by keyword. We send a confirmation email; one click activates.

  2. We ship at 7 a.m.

    One nightly fetch to the arXiv API for the union of every subscriber's filter, then per-subscriber slicing. Titles and abstracts translated to Spanish via MyMemory; originals one link away.

  3. You reply with numbers.

    Type 1 3 5, or "quiero el 2 y el 7", or "the second one". The listener parses the integers, fetches those PDFs from arXiv, and replies with them attached — usually within thirty seconds.

  4. Manage anytime.

    Every digest carries a signed link to edit categories, change the daily cap, or unsubscribe. No login, no password. The link is yours alone.

04 — Frequently asked

four questions, four answers
What does it cost?
Nothing. The whole stack runs on Oracle Cloud's always-free tier (one shared CPU, 1 GB RAM, six Docker containers in ~165 MB idle), plus free quotas of Brevo for outbound mail and Mailsac for reply parsing. If it ever gets popular enough to outgrow that envelope we'll add a Pro tier for power features — but the basic digest stays free, forever.
The reply-to-PDF thing — how does that actually work?
Every digest is sent with Reply-To pointing at a Mailsac inbox we poll every 10 minutes. When a reply arrives, we identify you by your From address, pull up your snapshot of that morning's edition, extract integers from the body (any natural-language wrapper is fine), download those PDFs from arXiv, and reply with them attached. Senders we don't recognise are silently dropped.
Who's behind this?
An indie open-source project. Source on GitHub, MIT licensed, self-hostable end-to-end. Originally built as a university Cloud Computing project; now run as a public service. No company, no investors, no ads, no data resold.
How do I cancel?
Every email has a one-click unsubscribe link in the footer, plus a List-Unsubscribe header so Gmail's native button works too. You can also reply with STOP, or write to the address shown in the colophon.
Colophon Set in Georgia, with system sans for interface chrome and a monospaced stack for code and paper identifiers. Sole accent: amber #D99C5E. Built and served on an always-free VM. The masthead is set 14% letterspaced because newspapers do that and it works.
